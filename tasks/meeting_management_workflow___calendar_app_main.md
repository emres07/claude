## Description
Core workflow for creating, updating, and managing clinic meetings

Workflow Steps:
- User selects date and time for meeting
- System validates meeting slot availability
- User enters meeting details (title, description, attendees)
- System saves meeting to database
- Meeting appears in calendar view
- User can edit or delete meeting
